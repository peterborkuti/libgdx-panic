Libgdx-panic

A retro game is written with LibGdx inspired by Panic for C-64 which was written by Ian Gray

Download this project as a .zip file Download the runnable desktop jar

Welcome to Libgdx Panic C64 game page

You can try it on web on this link

Description

Your world consists of floors and ladders. Enemies are chasing after you. They can be eliminated only if they are falling at least as much floors as the number of level. You have some timed bombs which create a hole in the floor. You can activate a bomb with SPACE and you can move with the ARROWS.

Running the jar file Download the runnable desktop jar

  1. Download the jar file
  2. java -jar desktop-1.0.jar
